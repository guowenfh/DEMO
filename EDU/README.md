## 思考
- 课程列表每一次在切换时，都全部移除，再创建,应该保留布局只切换内容。(性能优化，我不清楚到底怎么样好)思考了两种方法。
 - 使用innerHTML。布局部分用字符串写死，内容部分设为变量，在页码切换时，更换请求参数，返回的值动态修改变量值，并局部刷新页面。
 - 维持现在使用使用创建DOM节点的方式，但在页码切换时，循环获取每一个需要改变节点的属性或文本等等，切换时再循环重新赋值。
